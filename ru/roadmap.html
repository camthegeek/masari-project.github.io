Дорожная карта Masari

Эта дорожная карта управляется сообществом и может быть изменена, представляет собой предыдущие достижения (неполные) и предполагаемые сроки для будущих планов.

Q4 2017

WHM Алгоритм Сложности

Необходимое обновление, реализующее один из самых известных алгоритмов в порядке смягчения проблем с флеш-майнингом в качестве нового альткоина, с помощью pull request сделано в первую очередь в качестве вклада в Monero.

Q4 2017

GUI Кошелек

Предполагалось, что новый, простой в использовании кошелек GUI будет доступен, но были наложены временные ограничения на портирование одного из Monero для использования, пока эта инициатива сливается с целью мобильного кошелька.

Q1 2018

Поддержка Мультиподписей и Субадресов

Обе вышестоящии функции. Мультиподпись позволит осуществлять недоверенные транзакции вне цепочки, которые в противном случае потребовали бы контрагента. Субадреса позволяют одному закрытому ключу Masari выдавать несколько субадресов, выходя за пределы ID платежей.

Q2 2018

CryptoNight-Fast PoW

Мы создали наш собственный вариант CN, для того чтобы обеспечить дополнительную защиту от злонамеренных сетевых атак, при этом будучи более стабильным и дружественным к CPU.

Q3 2018

Веб и Мобильный Кошелек

Это нецелено на массовую адаптацию, включая поддержку мобильных устройств для повседневного использования. Клиенский веб-кошелек можно найти на здесь и Android кошелек можно найти на Google Play Store

Q4 2018

Uncle Mining (SECOR)

Эта уникальная для Masari функция обеспечивает более безопасную сеть, быстрые блоки и вознаграждение майнеров за осиротевшие блоки.

Q4 2018

Угол Сотрудничества

Это социальный протокол, разрабатываемый в качестве руководства для поощрения вклада и децентрализации управления проектом.

2019

Белая Бумага Blocktree

В ожидании дальнейших исследований, которые исследуют потенциал сегментирования PoW протокола CryptoNote, посредствам использования преимущества одноразовых выходов открытого ключа для разделения / сегментирования транзакций в масштабируемое Blocktree.

2019

Интеграция с Ledger

Поскольку многие предпочитают аппаратные решения для хранения MSR, Ledger должен поддерживаться в качестве альтернативы хранения.

Simple Private Tokens (SPT)

Без смарт-контрактов это легко дает возможность выпускать частные цифровые активы в сети конечному пользователю, с любой вычислительной логикой с использованием взаимозаменяемой архитектуры вне цепи.

Реализация Blocktree

В ожидании предшествующего исследования, реализация должна быть завершена к концу года.
